About Ian Belicina

Transitioning from a background in data analytics, my interest in software engineering was sparked during my undergraduate studies in mathematics. I was initially drawn to mathematics for its logical problem-solving aspects, and I soon recognized a similar problem-solving dimension in coding. Initially, I saw software engineering as a tool to complement my mathematical skills, but I quickly realized it's much more than that. I came to understand that software engineering is about crafting efficient, user-centric solutions that can have a meaningful impact on people's lives. This shift in perspective has driven my passion for software engineering, leading me to continually refine my skills, stay updated with emerging technologies, and adopt best practices.

In my career, I aim to merge the analytical precision developed in data analytics with the creative problem-solving capabilities found in software engineering. My goal is to create software solutions that are both technically robust and user-friendly. I'm committed to contributing my unique blend of skills to innovate and create software.

inclusiTravel

Tech Stack: Python | FastAPI | PostgreSQL | JavaScript | React | HTML | CSS | Bootstrap

InclusiTravel is a forward-thinking project that utilizes advanced technologies like Python, FastAPI, React, and Bootstrap to promote inclusivity. It empowers users, especially those with disabilities, to make informed choices about accessible venues. Our goal is to create a more inclusive world by providing easy access to comprehensive venue information through secure APIs while ensuring a user-friendly experience. Moreover, InclusiTravel is designed to be ADA-friendly, ensuring that it meets the accessibility standards set forth by the Americans with Disabilities Act, guaranteeing an inclusive experience for all.

WeatherNow

Tech Stack: Python | Django | PostgreSQL | JavaScript | React | HTML | CSS | Bootstrap

WeatherNow is a dynamic web application that seamlessly brings real-time weather information to your fingertips with a dash of user-friendly interactivity. This feature-rich platform is crafted using a versatile technology stack, including Python, Django, JavaScript, React, and HTML. It empowers users to access up-to-the-minute weather conditions for any location, displayed in a visually pleasing card format. Beyond the basics, WeatherNow also allows users to add locations to their list and even delete them, ensuring a tailored weather experience. Leveraging the integration of a third-party API for weather data, WeatherNow ensures accuracy and reliability in every forecast.
